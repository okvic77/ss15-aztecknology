matchesSelector helper

matchesSelector is pretty hot :fire:, but has vendor-prefix baggage :handbag: :pouch:. This helper function takes care of that, without augmenting Element.prototype.

matchesSelector( elem, selector );

matchesSelector( myElem, 'div.my-hawt-selector' );

// this DOES NOT polyfill myElem.matchesSelector

Package managers

Install with :bird: Bower

bower install desandro/matches-selector

Install with :nut_and_bolt: component

component install desandro/matches-selector
